Overview

Mobile application development, rich internet applications, user experience and big data are among our priorities; currently, GFT Appverse has published two key assets to advance with our open innovation objectives;

  • Appverse Mobile – A multi-platform mobile development framework allows developers to use HTML5, JavaScript, CSS3 and to leverage native functionalities of a mobile device to build apps from a common codebase.
  • Appverse Web – A multi-frontend technology capable web application framework incorporating the best-of-breed open source stack to meet the challenges of secure and robust server-side service development, integration and delivery.

We welcome individuals and organizations to accompany us in this journey that we believe to be essential for us all.

For more information about these projects, please refer to either Appverse Mobile or Appverse Web project sites. For more information about The Appverse Project as a whole, please refer to our informational site. For information about contributing with the project, please refer to the Appverse Public License page and the contribute page on our site.
